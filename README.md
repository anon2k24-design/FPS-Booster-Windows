# FPS Booster Bundle

Game-specific Windows optimization presets for popular titles.

## Supported games

- Apex Legends
- Fortnite
- Call of Duty
- Minecraft
- Dead by Daylight

## Features

- Disables Fullscreen Optimizations for supported games.
- Applies game-specific performance settings.
- Helps reduce stutter and latency.
- Provides a simple menu-based workflow.

## How to use

Run PowerShell as Administrator:

```powershell
.\fps-booster-bundle.ps1
```

Then choose the game you want to optimize.
